title: Manage configuration
---
This command is used to manage configuration of Hexo.

``` bash
$ hexo config [key] [value]
```

## List

``` bash
$ hexo config
```

### Get 

``` bash
$ hexo config <key>
```

### Set 

``` bash
$ hexo config <key> <value>
```

When `value` is `null`, the `key` will be deleted.
